indexOf (tbl, value)
Searches through a table and finds the first key with the given value.

Parameters:

  • tbl The table to search.
  • value The value to seach for.

Returns:

    the key of the value if the value exists. nil otherwise.

Raises:

type error - if tbl is not a value

Usage:

    local k = tableUtils.indexOf(tbl, 9)

range (tbl, start, finish)
Copies part of a table to a new table starting with the key 1.

Parameters:

  • tbl The table to copy.
  • start The point to start copying from.
  • finish The point to copy upto.

Returns:

    A copy of the table containint the elements from tbl[start] to tbl[finish].

Raises:

type error - if tbl is not a table
raise type error - if start is not a number
raise type error - if finish is not a number

Usage:

    local newTbl = tableUtils.range(tbl, 4, 8)

combine (tbl1, tbl2)
Combine two tables together into a new table. The first table will be given keys 1 to n and the second will be given kets n + 1 to n + m

Parameters:

  • tbl1 A table.
  • tbl2 Another table.

Returns:

    A combination of both tables.

Raises:

type error - if tbl1 or tbl2 is not a table

Usage:

    local tbl3 = tableUtils.combine(tbl1, tbl2)

binarySearchInternal (tbl, searchFor, low, high)
Searches through a table and finds the first key with the Given Value. The table must be sorted in accending order. If multiple keys have the same value there is no defined behaviour for wich key is returned.

Parameters:

  • tbl The table to seach.
  • searchFor The value to search for.
  • low The key to search from.
  • high The ket to search to.

Returns:

    the key of the value if the value exists. nil otherwise.

Usage:

    local k = binarySearchInternal(tbl, 9)
