Functions

split (str, sep)
Splits a string by a seperator.

Parameters:

  • str The string to split.
  • sep The seperator.

Returns:

    A table of strings. Created from the largest substrings of str which do not contain sep.
trimLeadingSpaces (str)
Removes leading spaces from a string.

Parameters:

  • str The string.

Returns:

    A substring of str with any spaces before the first non space character removed.

Usage:

    local noSpaces = textUtils.trimLeadingSpaces(str)
trimTrailingSpaces (str)
Removes trialing spaces from a string.

Parameters:

  • str The string.

Returns:

    A substring of str with any spaces after the last non space character removed.

Usage:

    local noSpaces = textUtils.trimLeadingSpaces(str)
toKey (str)
Converts a string to a value that could be used to index a table using dot notation. Spaces are replaced with underscores. the string is truncated after the first dot. For example "a b.c" could not be used as a key to index a table so the string is edited to "a_b".

Parameters:

  • str A string.

Returns:

    a string that can be used to index a table using dot notations following the rules stated above.

Usage:

    local key = textUtils.toKey(str)
toWords (str)
Temp function

Parameters:

  • str
trim (str, length)
Takes a string and trims it down to to the as many whole words it can fit inside the specified length. Returns the trimmed string and the rest of the discarded string as seperate returns value.

Parameters:

  • str A string.
  • length The length to trim the string down to.

Returns:

  1. The string truncated upto the length specified including only whole words.
  2. The part of the string truncated, without leading whitespace

Usage:

    local words = textUtils.trim(str, 10)
wrap (str, width, height)
Text wraps a string into a given height and width and handles '\n'. Wraps the string over multiple lines only spliting the line between words or during a word if it is larger than the line.

Parameters:

  • str The string to wrap.
  • width The width of the wrapped text.
  • height the height of the wrapped text.

Returns:

    A text wrapped version of the input string.

Usage:

    local wraped = textUtils.wrap(str, 20, 10)
pop (str, n)
Pops n characters off the ends of a string and returns both parts.

Parameters:

  • str The string.
  • n How many characters to pop. Default is 1.

Returns:

  1. The string truncated by n characters.
  2. The truncated characters.

Usage:

    local str, last = textUtils.pop(str)

Local Functions

wrapInternal (str, width, height, lines)
Text wraps a string into a given height and width. Wraps the string over multiple lines only spliting the line between words or during a word if it is larger than the line.

Parameters:

  • str The string to wrap.
  • width The width of the wrapped text.
  • height the height of the wrapped text.
  • lines A table of lines that is appened by this function.

Returns:

    A text wrapped version of the input string, appened on to lines.

Usage:

    local wraped = wrapInternal(str, 20, 10, {})
