Functions

isInt (x)
Checks whether a number is an integer.

Parameters:

  • x A number.

Returns:

    true if the number is an integer.

Raises:

type missmatch error - if x is not a number

Usage:

    local isInt = mathUtils.isInt(4.3)

div (x, y)
Preforms integer divison on x by y.

Parameters:

  • x The Dividend.
  • y The Divisor.

Returns:

    The result of x divided by y truncated towards 0.

Raises:

type missmatch error - if x or y is not a number

Usage:

    local a = mathUtils.div(5, 2)
time (func, trials)
Times the time it takes for a function to complete.

Parameters:

  • func The function to time.
  • trials The number of times to call the function.

Returns:
