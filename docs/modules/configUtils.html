Module configUtils

Allows for easy editing of config files stored in json format.

Json files mat be easily created, read and edited.

Functions

configEditor (path, readable)
Creates a configEditor object that can be used to manipulate json files.

Parameters:

  • path The path to the json file.
  • readable wether or not the json file should be saved in human readable format.

Returns:

    A configEditor object.

Usage:

    local editor = configUtils.configEditor(path, false)
self.get (k)
Gets a value from the config file.

Parameters:

  • k The key of the valie.

Returns:

    The value from the config file with key k.

Usage:

    local value = config.get(4)
self.set (k, v)
Sets a value in the config file.

Parameters:

  • k the key to write at
  • v the value to write to the config file

Usage:

    config.set(4, "9")

Saves changes and writes them back to the files
