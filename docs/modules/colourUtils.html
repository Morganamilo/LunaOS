Module colourUtils

The colourUtils API provides function to manipulate both colours and blits and converting between the two.

Functions

colourToBlit (colour)
Converts a colour to a blit.

Parameters:

  • colour A colour.

Returns:

    The blit corresponding to the given colour. If a valid blit is entered then return the blit back. Otherwise return nil.

Usage:

    local red = colourToBlit(colourUtils.colours.red)
blitToColour (b)
Converts a blit to a colour.

Parameters:

  • b A blit.

Returns:

    The colour corresponding to the given blit. If a valid colour is entered then return the colour back. Otherwise return nil.

Usage:

    local red = blitToColour(colourUtils.blits.red)
lighter (colour)
Gives a lighter version of a given colour. White will always return while as it can not get any lighter.

Parameters:

  • colour The colour to make lighter.

Returns:

    a lighter version of colour.

Usage:

    lighterRed = colourUtils.lighter(colourUtils.blits.red)
darker (colour)
Gives a farker version of a given colour. Black will always return black as it can not get any darker.

Parameters:

  • colour The colour to make darker.

Returns:

    a darker version of colour.

Usage:

    darkRed = colourUtils.darker(colourUtils.blits.red)
invert (colour)
Gives the inverse of a given colour. Inverting a colour twise it not guaranteed to give the original colour.

Parameters:

  • colour The colour to be inverted.

Returns:

    The inverse of the colour.

Usage:

    invertRed = colourUtils.invert(colourUtils.blits.red)
