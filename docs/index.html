Modules

object The object module allows the creations of classes, objects and interfaces.
kernel The kernel allows multiple processes to be created from either functions, files or programs Only one process can run at a time so each process runs then pauses to let another process run.
windowHandler The windowHandler is responsable for managing the window each process draws to.
