Class Component

Component is the base to all GUI components that can be added to a frame/view.

It implements the drawable and eventHandler

Tables

Component.listeners
The events that the Component is listening for. It hold the event name as its key and the function to be called when that event happens as its value.

Fields

Component.active
Whether the component is active. If a component is not active it can not recive events.
Component.visible
Whether the component can be seen. If a component is not visable it will not be drawn and can not recive events.

Methods

Component:handleEvent (data)
Hanles an incoming event. Only handls the event if the component is both visable and active. It calls its lisreners and passes itself and the unpacked event through as an argument.

Parameters:

  • data an event.

Usage:

    component:handleEvent(event)
Component:addEventListener (event, func)
Adds an event listner to the listeners tables along with the function to call when the event happens.

Parameters:

  • event The event to listen for.
  • func The function to be called when the event fires.

Usage:

    component:addEventListener("key", function(event, key, held) print(key .. " is pressed"))
Component:removeEventListener (event)
Removes an event from the listeners table.

Parameters:

  • event The event to remove.

Usage:

    component:removeEventListener("key")
Component:onDraw (buffer)
Calls the Component:draw method but only if the component is visable.

Parameters:

  • buffer The buffer passed to Component:draw

Usage:

    component:onDraw(buffer)
Component:draw (buffer)
Draws the component to a buffer. This method is empty and is desinged to be overriden by subclasses.

Parameters:

  • buffer

Usage:

    component:draw(buffer)
Component:getFrame ()
Gets the frame the component is placed on. Will fail if the comonent is not placed on a view/frame

Usage:

    local frame = component:getFrame()
Component:getParentPane ()
Gets the pane that the component is placed on. Will fail if the comonent is not placed on a view/frame

Usage:

    local parentPane = component:getParentPane()
Component:setCursorPos (xPos, yPos)
Sets the cursorPos of the frame

Parameters:

  • xPos
  • yPos

Usage:

    component:setCursorPos(3, 4)
Component:getCursorPos ()
Gets the cursorPos of the frame

Returns:

    x and y coordinates of the parent pane.

Usage:

    local x, y = components:getCursorPos()
Component:setCursorBlink (blink)
Sets the cursor blink of the frame. The cursor either blinks is either enabled or disabled there is no inbetween.

Parameters:

  • blink true for the cursor to blink, false to make it not blink.

Usage:

    component:setCursorBlink(true)
Component:isInBounds (xPos, yPos)
Gets whether an x and y coordinate is inside the bounds of the component. this method is empty and is designed to be overriden by subclasses.

Parameters:

  • xPos The x position of the coordinate.
  • yPos The y position of the coordinate.

Returns:

    true if the xPos and yPos are inside the component.

Usage:

    local inBounds = component:isInBounds(5, 7)
Component:setParentPane (pane)
Sets the parent pane of the component. The parent pane will usually be a frame/view.

Parameters:

  • pane The pane to set as the components parent pane.

Usage:

    = component:setParentPane(frame)
Component:setCursorColour (colour)
Sets the cursorColour of the frame. Will fail if the component is not placed on a pane.

Parameters:

  • colour The colour to set the cursor

Usage:

    component:setCursorColour(colourUtils.blits.blue)
Component:getAbsolutePos ()
Gets the absolute position of the component. Normaly the x and y position stored in a compoenent is relative to its pane. This method will always get the position relative to the frame insead of the parentPane. This method will fail if it is not placed on a pane.

Returns:

    x coordinate relative to the frame, y coordinate relative to the frame

Usage:

    local xPos, yPos = component:getAbsolutePos()
Component:requestFocus ()
Requests focus from the frame. This will cause every other component to lose focus. This method will fail if the component is not placed on a pane.

Usage:

    component:requestFocus()
Component:unFocus ()
Unfocuses the component. Will only unfocus itslef and do nothing if another component is focused. This method will fail if it is not placed on a pane.

Usage:

    component:unFocus()
Component:isFocus ()
Gets whether or not the comonent is focued.

Returns:

    true if the component is focued.

Usage:

    local isFocused = component:isFocus()
