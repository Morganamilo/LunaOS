Script recovery

The LunaOS recovery provides some features outside of the actual OS.

LunaOS recovery supports the following features:

  • Reboot System: Reboots the computer
  • Use Craft OS: Instead of booting LunaOS the default OS it ran instead
  • Fix permisions: Set file Permision back to their default
  • Wipe Data: Wipes all the program dada inside of LunaOS
  • Uninstall: Uninstall the OS completley
Information onscreen is displayed using a menu driven interface. The interface is navigated using the arrow keys and enter.

Local Functions

writeCenter (str, line, colour, backColour)
Writes a string to the center of the screen on a given line with a given colour and text colour. The text is centered while the background colour of the text spans the entire line.

Parameters:

  • str The string that is written to the screen.
  • line The lime that the string is written to. 1 is the top of the terminal n is the bottom where n >= 1.
  • colour The colour of the string using ComputerCraft's colours API.
  • backColour The background colour of the string using ComputerCraft's colours API.

Usage:

    writeCenter("Hello World", 5, colours.white, colours.blue)
writeDivider (line, colour)
Draws a horizontal divider across the sreen on a given line and in given colour. The divider is made up of a repitition of the character "-" and is in the colour specified. The background colour remains unchanged.

Parameters:

  • line The line that the divider is drawn to.
  • colour The colour that the divider is drawn in using ComputerCrafts colours API.

Usage:

    writeDivider(5, colours.white)
writeDesc ()
Writes the current description to the screen. _secondaryTextColour is used for the textColour.

Usage:

    updateDesc()
writeOptions ()
Writes all the options to the screen. The first option is written on line 3, each following option is written on the following line downward. _secondaryTextColour is used for the text colour. if the option it is writing is selected _selectedColour is used as the backgroundColour. otherwise _backColour is used instead effecivley making it invisable.

handleOption ()
Executes the currently selected option. Sets the decription to "Completed" if the option completed successfully Otherwise the description is set to the error that occured.

Returns:

    true if the currently selected options name was "craftos", false otherwise.

Usage:

main ()
The main loop that polls for input and act accordingly. Initially sets the description. Moves the selected option up and down using arrow key input. Executes the currently selected option when enter is pressed.
